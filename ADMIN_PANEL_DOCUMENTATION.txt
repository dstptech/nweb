PROJECT OVERVIEW
================================================================================
The DSTP Admin Panel is a comprehensive administrative dashboard built with React
and Vite, designed to manage the DSTP Website's content and operations. It provides
an intuitive interface for administrators to edit website content, manage services,
projects, blog posts, and view analytics dashboards.

Tech Stack:
  • React 19.2.0 - UI library
  • Vite 5.1.0 - Build tool and dev server
  • React Router DOM 7.13.1 - Client-side routing
  • Tailwind CSS 4.2.1 - Utility-first CSS framework
  • Axios 1.6.0 - HTTP client for API requests
  • React Hot Toast 2.6.0 - Toast notifications
  • Recharts 3.8.0 - Chart library for analytics
  • Lucide React 0.577.0 - Icon library
  • ESLint 9.39.1 - Code linting

Base URL: http://localhost:5173 (Vite dev server)
API Base URL: http://localhost:8000 (Django backend)

API INTEGRATION DETAILS
================================================================================

Base Configuration:
  URL: http://localhost:8000
  Headers: Content-Type: application/json
  Credentials: Include (withCredentials: true)

Authentication:
  ├─ Token Storage: localStorage.getItem('access_token')
  ├─ Token Injection: Authorization header (Bearer token)
  ├─ Error Handling: 401 Unauthorized (token expired)
  └─ Ready for: Token refresh mechanism

Endpoints Used:
  1. Homepage Data Management
     ├─ GET /api/v1/homepage/homepage/
     │  └─ Fetch all homepage data (returns array, take first item)
     ├─ POST /api/v1/homepage/homepage/
     │  └─ Create new homepage data
     ├─ PUT /api/v1/homepage/homepage/{id}/
     │  └─ Full update (replace all fields)
     ├─ PATCH /api/v1/homepage/homepage/{id}/
     │  └─ Partial update (only changed fields)
     └─ DELETE /api/v1/homepage/homepage/{id}/
        └─ Delete homepage data

Data Schema (Homepage):
  {
    id: number,
    hero: {
      hero_title: string,
      hero_subtitle: string,
      hero_image: string (URL),
      banner_text: string,
      banner_image: string (URL),
      stats_title: string,
      stats_value: string
    },
    services: array,
    projects: array,
    testimonials: array,
    about: {
      title: string,
      subtitle: string,
      description: string,
      vision: string,
      stats: array
    },
    contact: {
      title: string,
      subtitle: string,
      btnText: string,
      btnLink: string,
      email: string,
      phone: string,
      address: string,
      bgColor: string (hex),
      textColor: string (hex)
    }
  }

================================================================================
TROUBLESHOOTING
================================================================================

Common Issues:

1. Vite Dev Server Not Starting
   Solution: Kill process on port 5173 or change port in vite.config.js

2. API Connection Errors
   Ensure Django backend is running on http://localhost:8000

3. CSS Not Loading Properly
   Clear node_modules and reinstall: rm -rf node_modules && npm install

4. Hot Module Replacement (HMR) Not Working
   Check firewall settings or restart dev server

5. Token Expiration Issues
   Clear localStorage and login again: localStorage.clear()
